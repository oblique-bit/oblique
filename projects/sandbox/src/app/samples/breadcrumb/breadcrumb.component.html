Breadcrumb

Demo

Try out the links below to see different features of the breadcrumb component in action.

Here's a breadcrumb component with default configuration:
<ob-breadcrumb></ob-breadcrumb>

Here's a breadcrumb component with a max-width configuration:
<ob-breadcrumb maxWidth="8ch"></ob-breadcrumb>

This doesn't work in Firefox: Here's a breadcrumb component with a max-width configuration of 2ch:
<ob-breadcrumb maxWidth="2ch"></ob-breadcrumb>

This works in Firefox: Here's a breadcrumb component with a max-width configuration of 3ch:
<ob-breadcrumb maxWidth="3ch"></ob-breadcrumb>

The max width property can be freely set by consumers of the component. It should be noted that very small width values expressed in character units such as 1ch or 2ch do not reliably produce overflow in Firefox. Firefox calculates the minimal text width differently from Chromium in these boundary cases and will not generate visible overflow. Consumers are therefore advised not to use max width values below 3ch if consistent ellipsis behavior is required in Firefox. This is a technical characteristic of the Firefox layout engine.

Here's a breadcrumb component with a max-width configuration of 2rem:
<ob-breadcrumb maxWidth="2rem"></ob-breadcrumb>

Here's a breadcrumb component with url beautifing disabled (only applies to routes with no label and parameter values):
<ob-breadcrumb [beautifyUrls]="false"></ob-breadcrumb>

Here's a breadcrumb component with a custom separator (only applies to routes with no label and more than one part in the path):
<ob-breadcrumb parameterSeparator=" | "></ob-breadcrumb>
