Reduce your browser's window size in order to see the section's background color changing from red to blue. Or increase it for the opposite effect.

The ob-media-breakpoint-down mixin is used to define CSS rules that are only applied when the size of the viewport is smaller than the value passed as parameter (in this example, the md value).

The viewport is the portion of the browser's window where the application is rendered.

Red background for large screens
