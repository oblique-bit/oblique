The most important files to look at in the StackBlitz Translations example are:

  • app.module.ts
  • ./translations/translations.component.ts
  • ./translations/translations.component.html
  • ../assets/i18n/es.json
The oblique-*.json files in the ../assets/i18n folder only needed to be added for the translations to work on StackBlitz. Do NOT do this in your own project, this is not necessary for Oblique translations to work correctly in a typical project.
