A component and related directive to simplify nesting of forms. Forms can also be nested within nested forms. They can be integrated in 2 simple steps:

  1. Create the sub-form
    • a simple component with a form (without implementing either ControlValueAccessor or Validator)
    • set the exportAs property in the component's decorator
    • template driven forms also need to expose NgForm via a ViewChild
  2. In the parent's component template
    • With Reactive form
      • add the orParentForm directive on the parent form, i.e. the element with the formGroup directive
      • wrap you sub-form with the or-nested-form component
      • pass the sub-form's formGroup to the or-nested-form component's nestedForm input using the exportAs property.
    • With template driven form
      • add the orParentForm directive on the form element
      • wrap you sub-form with the or-nested-form component
      • pass the sub-form's NgForm.control to the or-nested-form component's nestedForm input using the exportAs property.
