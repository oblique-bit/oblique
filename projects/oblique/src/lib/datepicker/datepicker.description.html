Wrapper for the ng-bootstrap Datepicker component with some controls (clear, open & close). The validation is left to the containing form

Deprecated

This component is meant to be used with Bootstrap. Material theme should instead use the Angular implementation demonstrated below.

As a partially deprecated component, it won't be prioritized for new features or improvements but won't be removed as long as Bootstrap is supported.

Do NOT apply form-control class on or-date-picker as it will break the layout.

Deprecated with Material Design

This component is only deprecated with Material Design. With Bootstrap theme, it can and should be used.

As a partially deprecated component, it won't be prioritized for new features or improvements but won't be removed as long as Bootstrap is supported.
